if you have permission for tp: <br>
/tp (teleport to coords)<br>
/tp2wp (teleport to waypoint)<br>
/tp2player (teleport to player by using playername)<br><br>

all users:<br>
/calladmin<br><br>

in planning:<br>
ghostmode (noclip)<br>
godmode<br>
set health 100%<br>
trolling (e.g. freeze)<br><br>
